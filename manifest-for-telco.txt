Upgrade SAP Commerce Cloud version and SAP Commerce Cloud, Telco and Utilities Accelerator version in the manifest.json file to initiate the upgrade process:

{
       "commerceSuiteVersion" : "2211",
       "extensionPacks" : [
       {
              "name" : "hybris-b2ctelco-suite",
              "version" : "2302.8"
       }
       ],
       ...
}

Reference:
